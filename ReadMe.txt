Prerequisites:
NodeJS, Angular cli, dotnet core 2.1


Run Application:
-Open command prompt and point to RateCalculationEngine folder.
-Run command: dotnet run
-Open command prompt and point to ClientApp folder.
-Run command: ng s -o

Alternative access to application:
-I have hosted this in AWS. (Front end angular into S3 static website hosting and dotnet core webapi into lambda)
-Use this url to access this application: http://parking-rate-calculator.s3-website-ap-southeast-2.amazonaws.com/
